# Hyperledger Fabric

# build docker image
```bash
clone https://github.com/Yunpeng-J/HLF.git $GOPATH/src/github.com/hyperledger/fabric
cd $GOPATH/src/github.com/hyperledger/fabric
make native
make docker
```

# Run locally

```bash
cd local
# see README.md
```

# Run in docker swarm 
```bash
cd docker
# see README.md
```
