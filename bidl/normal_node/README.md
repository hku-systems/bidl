# BIDL Normal Node

Clone repo
```shell
git clone git@github.com:lililqth/normal_node.git $GOPATH/src/github.com/lililqth/normal_node
```

## How to run 
go to the root directory of the project

```shell
go run ./cmd/client 
go run ./cmd/server
```
